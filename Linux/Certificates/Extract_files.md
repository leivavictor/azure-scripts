# Extract files in linux

## TAR files

### .tar.gz

Command: tar

-x: extract files from a tar file.

-z: Indicates that the file is gzip.

-f: to specify the file.

Example:

```bash
tar -xzf %filename%.tar.gz
```

### .tar.xz

Command: tar

-x: extract files from a tar file.

-f: to specify the file.

Example:

```bash
tar -xf %filename%.tar.xz
```
